## 🚀 Projetos em Destaque

### 🌌 **[Projeto Aurora (Document Intelligence)](#)**
Sistema de apoio à decisão que utiliza **LLMs** para automatizar a extração de dados e validação de documentos acadêmicos. O projeto foca em transformar fluxos manuais em processos semi-automáticos, onde a IA gera pareceres técnicos a partir de arquivos brutos (CSV/Imagens) para validação humana.
*   **Tech:** Python, LLMs, OCR, Data Pipelines.

### 📊 **[Otimização de Modelos (Fine-tuning)](#)**
Repositório dedicado a experimentos com **LoRA** e **PEFT** para especializar modelos de linguagem em domínios técnicos específicos, focando em redução de custo computacional e ganho de performance.

### 🤖 **[Agente Conversacional RAG](#)**
Implementação de um pipeline completo de RAG para consulta de bases de conhecimento privadas, utilizando **LangChain** e bancos de vetores para respostas contextuais precisas.
